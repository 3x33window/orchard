Grave of  The First Dead Tree

This tree secreted red sap when itself was wounded.
The secreted red sap solidifies and fills up the part of the wounded tree's body.

이 나무는 줄기에 상처를 입으면 붉은 수액을 분비했다.
나무는 분비한 수액을 굳혀 갈라진 몸의 일부를 메꾸며 살아남았다.

This tree was excessively injured, so continuosly secreted red sap to protect itself from injuries.

이 나무는 과도한 상처로 인해, 계속해서 붉은 수액을 분비했다.

Because of this, the original body of the tree was lost and only the hardened sap remained.

때문에 나무 본래의 몸을 모두 잃고 굳은 수액만 남아버렸다.

It was after a long time, that We realized the death of this tree.

우리는 아주 오랜 시간이 지나고서야 이 나무를 잃었음을 알게되었다.
